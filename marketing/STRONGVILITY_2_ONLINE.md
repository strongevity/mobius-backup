# SECTION 1: PROGRAMMING FOR ONLINE

## What Changed from In-Person

### Key Differences
- **No real-time cueing** — You won't see them during the set
- **Video check-ins** — Clients film and send via TrueCoach
- **Self-assessment focus** — Clients must learn to feel correct movement
- **Simpler movements** — Fewer complex lifts requiring immediate correction
- **Higher accountability** — Clearer check-in protocols essential

### Programming Adjustments
- Prioritize movements with wide margin of error
- More regressions readily available
- Clear self-check criteria before advancing
- Weekly video submission required
- Program designed for 80% autonomous completion

# SECTION 3: SCALING (Online-Optimized)

### Tier 1: FOUNDATION (New to Online)
- Very simple movements
- High confidence in self-assessment
- Weekly video required
- More frequent check-ins (2x/week)

### Tier 2: DEVELOPMENT (Comfortable with Online)
- Moderate complexity
- Reliable self-assessment
- Weekly video sufficient
- Standard check-ins (1x/week)

### Tier 3: PERFORMANCE (Independent)
- Can handle more complex cues
- Excellent form over time
- Bi-weekly video OK
- Monthly consultation

# SECTION 4: PROGRESSION TRIGGERS (ONLINE)

## The Video Submission Protocol

### Every Week, Client Must Submit:
1. **One working set** of each main movement (filmed)
2. **Self-assessment:** Rate form 1-10 with explanation
3. **Pain check:** Any discomfort? Where?
4. **Question:** One thing to improve

### Progression Decision Tree

```
FRIDAY: Client submits videos
   ↓
YOU REVIEW: Watch each video
   ↓
FORM ISSUES? ──YES──→ Send correction, stay at weight
   ↓
NO
   ↓
ALL MOVEMENTS ≥ 8/10? ──NO──→ Same weight next week
   ↓
YES
   ↓
3 WEEKS IN A ROW ≥ 8/10? ──NO──→ Add weight or reps
   ↓
YES
   ↓
ADVANCE TO NEXT LEVEL
```

### What You're Looking For on Video

**Good Signs:**
- Full range of motion
- Controlled tempo
- Steady posture
- No visible grimace/pain
- Smooth transitions

**Red Flags:**
- Squinting/grimacing
- Movement substitution
- Range of motion loss
- Visible compensation
- Resting between reps longer than usual

# SECTION 8: CLIENT VIDEO GUIDE

### How to Film for Dave (Copy/Paste to Send)

**Camera Position:**
- Landscape mode (phone sideways)
- Far enough back to see full body
- Good lighting (face the light, not the window)

**What to Film:**
- Complete working set
- Pause at bottom/top of each rep
- Walk away from camera at end (show full gait for carries)

**Good Example:**
"Here's Tuesday's squat. 3x10 at 35 lb. Form felt solid, hit depth every rep."

**Bad Example:**
"Here it is" [blurry, 2 seconds, can't see feet or depth]

### Video Checklist (Send to Clients)
- [ ] Phone in landscape
- [ ] Can see entire body
- [ ] Good lighting (not backlit)
- [ ] Full range of motion shown
- [ ] Clear weight visible (if using equipment)

# SECTION 9: NO-PAIN PROTOCOL (ONLINE)

### Pre-Workout Self-Check
Client rates each body part 0-10:
- Shoulders: _
- Upper Back: _
- Lower Back: _
- Hips: _
- Knees: _

**If any 4-6:** Reduce load 20%
**If any 7+:** Skip exercise, substitute, message Dave immediately

### During-Set Rule
**3 Rep Rule:** If something feels off for 3 consecutive reps, STOP. Film it. Send it. Wait for response.

### Post-Workout (Every Session)
- Can you get up off the floor without using hands?
- Any new pain since starting?
- Did anything feel "off" that wasn't painful?

# SECTION 10: TOOL SELECTION BY VIDEO CLARITY

### Best Tools for Online Clients
- **KB** — Clear position, simple path
- **DB** — Easy to see weight, clear form
- **Trap Bar** — Great visibility, forgiving
- **Machine** — Fixed path, safe, consistent
- **Band** — Tension visible throughout
- **Bodyweight** — Easy to assess full ROM

### Tools Requiring Extra Attention Online
- **Barbell** — Complex path, may need multiple angles
- **Cable** — Need to show anchor point clearly
- **Monkey feet** — Only if client is very experienced

# SECTION 11: HANDOFF TOOLS

### When Client is Ready for In-Person
- Same programming logic applies
- Add load faster (you can correct in real-time)

### Upgrade Path
1. Online: All tiers, max bodyweight/machine
2. Online + KB: Introduce kettlebells
3. Online + DB: Add dumbbells
4. Online + Barbell: Introduce barbell variations
5. Hybrid: Online programming + periodic in-person
